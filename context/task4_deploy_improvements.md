# Task 4: Deploy Panel Improvements (10 Features)

## File Modified
- `static/dashboard.html`

## Features Implemented

### 1. Deploy History (Task 4.1)
- Stores last 5 deploy entries in `localStorage` key `deployHistory`
- Displayed below the deploy form in `#deployHistory` div
- Shows success/fail status, image, replicas, strategy, and timestamp
- Functions: `getDeployHistory()`, `saveDeployHistory()`, `renderDeployHistory()`

### 2. Image Search/Filter (Task 4.2)
- Search input `#imageSearchInput` above the image list
- Debounced (200ms) input handler filters `_lastImageData` array
- `renderImages()` now stores data in `_lastImageData` and respects search filter
- Shows count: "이미지 선택 (filtered/total)"

### 3. Image Size Display (Task 4.3)
- `formatImageSize(sizeMb)` function returns color-coded HTML
- Green (`img-size-green`): < 500MB
- Yellow (`img-size-yellow`): 500MB - 2GB
- Red (`img-size-red`): > 2GB
- Converts to GB for sizes >= 1024MB

### 4. One-Click Deploy Presets (Task 4.4)
- Preset buttons row: nginx, redis, postgres, httpd
- Clicking fills `#deployImage` and `#deployName` fields
- CSS class: `.preset-btn`

### 5. Deploy Progress Indicator (Task 4.5)
- Spinning CSS animation (`@keyframes spin`, `.deploy-spinner`)
- Shows spinner + "배포 중..." text during deploy API call

### 6. Per-Node Deploy Results (Task 4.6)
- Enhanced display with checkmark/cross icons per node
- Node name shown in bold, created count shown
- CSS class: `.per-node-result`

### 7. Image Pull Button (Task 4.7)
- Pull row with input `#pullImageName` and button `#btnPullImage`
- Calls `POST /v1/images/pull` with `{ image: ... }`
- Shows "Pulling..." state, reloads images on success

### 8. Auto-Generate Service Name (Task 4.8)
- `#deployImage` input handler extracts base name from image path
- Strips registry prefix and tag, fills `#deployName`
- Uses `data-autoGenerated` attribute to track auto vs manual input
- Stops auto-generating once user manually edits the name field

### 9. Replica +/- Buttons (Task 4.9)
- `#btnReplicaPlus` and `#btnReplicaMinus` around the replicas input
- Range: min 1, max 100
- CSS class: `.replica-adjust`

### 10. Deploy Template Save/Load (Task 4.10)
- Templates stored in `localStorage` key `deployTemplates`
- Save: prompts for name, stores all deploy form fields
- Load: fills all form fields from selected template
- Delete: removes selected template with confirmation
- Functions: `getDeployTemplates()`, `saveDeployTemplates()`, `refreshTemplateSelect()`
- UI: `#templateSelect`, `#btnTemplateSave`, `#btnTemplateLoad`, `#btnTemplateDelete`
